# MkDocs 中文文档翻译项目

# 说明

本文档为 [MkDocs](https://github.com/mkdocs/mkdocs/) 项目的中文文档翻译，本站并非中文官方网站（事实上也并不存在所谓的中文官方网站）。这里仅出于个人爱好进行文档翻译，为中国开发者提供阅读方便，文档翻译内容将以翻译内容准确无误为第一准则，建议有一定英语基础的开发者或使用者进行英文阅读。
